# Windows 自动部署工具 - 界面设计规范

## 一、设计原则

### 1.1 视觉风格

- **设计语言**：Material Design
- **色彩方案**：深色主题（Dark Theme），护眼且专业
- **字体**：Segoe UI（Windows 原生字体）
- **圆角**：8px 统一圆角
- **间距**：遵循 8px 网格系统

### 1.2 色彩规范

```
主色调：
- Primary: #1976D2 (蓝色)
- Primary Dark: #1565C0
- Primary Light: #42A5F5

强调色：
- Accent: #00BCD4 (青色)

背景色：
- Background: #121212 (深色背景)
- Surface: #1E1E1E (卡片背景)
- Dialog: #252525 (对话框背景)

文字色：
- Text Primary: #FFFFFF (87% 透明度)
- Text Secondary: #B3B3B3 (60% 透明度)
- Text Disabled: #666666 (38% 透明度)

状态色：
- Success: #4CAF50 (绿色)
- Warning: #FFC107 (黄色)
- Error: #F44336 (红色)
- Info: #2196F3 (蓝色)
```

### 1.3 布局规范

- **窗口尺寸**：900 x 650px（最小尺寸）
- **侧边栏宽度**：220px
- **内容区边距**：24px
- **卡片间距**：16px
- **控件间距**：12px
